Requirements
   

 
 
  • an HTML editor (WYSIWYG or plain-mode)
  • webspace or access to a computer with diskspace
    and a local installed (and hopefully configured) HTTP webserver.
  • perl interpreter (v. 5.6.0 or higher recommended)
  • patience & last but not least: real interest.
       
       
Sources    
       
    HTML editor:
      - Proton code editor v. 3.2 or higher
      - Notepad (on Win32 systems)
      - vi (on UNIX systems)
       
    HTTP webserver:
    - Apache HTTP webserver v. 2.0.44 or higher
       
    Perl interpreter:
      - Perl v. 5.6.0 or higher
